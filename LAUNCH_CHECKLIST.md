# 🚀 Launch Checklist - Earnings Scanner V2

## Step 1: Update Render Service

Go to https://dashboard.render.com and find your `earnings-scanner-api` service.

### Update Build & Deploy Settings:

**Build Command:**
```
pip install -r api/requirements.txt
```

**Start Command:**
```
cd api && uvicorn main_v2:app --host 0.0.0.0 --port $PORT
```

## Step 2: Add Environment Variables

In Render dashboard → Environment section, add:

1. `DATABENTO_API_KEY` = [your new key]
2. `SECRET_KEY` = any-random-string-for-jwt
3. `DATABASE_URL` = (auto-generated by Render if using PostgreSQL)

## Step 3: Deploy

Click "Manual Deploy" → "Deploy latest commit"

## Step 4: Test Your Live API

Once deployed, test these endpoints:

```
https://earnings-scanner-api.onrender.com/
https://earnings-scanner-api.onrender.com/api/opportunities
https://earnings-scanner-api.onrender.com/api/upcoming-earnings?days=7
https://earnings-scanner-api.onrender.com/api/test
```
