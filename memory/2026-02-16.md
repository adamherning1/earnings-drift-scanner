## Key Lessons Learned
1. **The original Keltner's edge is in exit management, not entry logic.** 0.5R TP with 82% WR is a scalping machine.
2. **Adding confirmation filters (MACD, squeeze, etc.) reduces trade count without proportionally improving quality.** Simplicity wins.
3. **Re-entry is the one enhancement that works** because it doesn't change entry quality — it just catches more of the same trends.
4. **Mean reversion fails on gold 4H.** Connors RSI, Bollinger bounce — all garbage. Gold trends too hard.
5. **Python backtest results don't match TradingView exactly.** Always validate in TV. Our Python showed $39K for re-entry but TV showed $23K.
6. **The original Keltner's "quantity over quality" approach is genuinely hard to beat.** Loose filters + tight TP + high frequency = compounding machine.
7. **Pine v5 works better than v6** for strategy scripts (v6 had commission type naming issues).
